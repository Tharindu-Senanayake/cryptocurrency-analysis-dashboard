Cryptocurrency Analysis Dashboard

Automated daily pipeline tracking 20 major cryptocurrencies in AUD - from API to interactive dashboard, fully unattended.

View Full Dashboard GitHub Repo

Key Skills Demonstrated

  • Data Pipeline Engineering - end-to-end automated ingestion to visualisation
  • Data Modelling - star schema with fact and dimension tables
  • Power BI - multi-page dashboard with desktop and mobile layouts
  • Python - production-quality scripts with logging and error handling
  • Workflow Automation - GitHub Actions daily cron, self-maintaining pipeline

Pipeline

▸ CoinMarketCap API
▸ Python ingestion script
▸ Append-only raw CSV
▸ Star-schema table builder
▸ GitHub Actions (daily, midnight UTC)
▸ Power BI live refresh

Tech Stack

Python pandas CoinMarketCap API GitHub Actions Power BI Star Schema CSV WAT Framework

Data Model

  • dim_coin - coin attributes (name, slug, max supply)
  • dim_date - date hierarchy (year, quarter, month, day)
  • fact_coin_prices - daily price, volume, market cap
  • fact_global_market - BTC dominance, DeFi, stablecoins
